# volumetric.js

Raycasting Volumetric Renderings for WebGL.


## API

  * setTexture(array, width, height, depth)
  * setExtent(minimum, maximum)
  * setSteps(value)
  * setOpacity(value)
  * setLighting(value)
  * draw()
